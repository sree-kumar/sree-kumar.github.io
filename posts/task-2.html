4 Automate

I have the code for automated exporting got from many sites. The most simplest one looks like this: 

(setq org-publish-project-alist
    '(("git" ;; my blog project (just a name)
       ;; Path to org files.
       :base-directory "~/tao-org/sree-kumar.github.io/org"
       :base-extension "org"
       ;; Path to Jekyll Posts
       :publishing-directory "~/tao-org/sree-kumar.github.io/posts/"
       :recursive t
       :publishing-function org-html-publish-to-html
       :headline-levels 4
       :html-extension "html"
       :body-only t
       :with-toc nil
       )))

For the time being this code is part of my emacs file. But Duncan MacVicar has self contained configuration which we will try at a later point of time.
